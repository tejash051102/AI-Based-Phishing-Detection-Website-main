PhishGuard AI

A full-stack phishing detection website that scans URLs, email text, and uploaded files using an AI prediction service, threat indicators, reputation checks, dashboards, reports, and admin monitoring.

React Node.js FastAPI MongoDB scikit-learn Enterprise Security

Company-Style Product Summary

PhishGuard AI is positioned as a security operations platform for teams that need fast phishing triage, evidence management, and risk visibility.

Business Problem

Organizations receive suspicious links, emails, and files every day, but manual review is slow and inconsistent.

Product Solution

PhishGuard AI provides AI-assisted scanning, explainable risk indicators, reports, alerts, and admin oversight.

Business Value

Security teams can reduce investigation time, centralize scan history, and prioritize high-risk threats faster.

Workflow

The main scan flow from user input to final report.

1
User submits URL, message text, or a supported file from the scanner page.
2
Express validates the request, parses uploaded content, and sends scan data to the AI service.
3
FastAPI extracts URL or text features and runs the phishing detection model.
4
Backend saves the verdict, score, indicators, reputation signals, logs, and notifications in MongoDB.
5
Frontend displays the result, explanation panel, scan history, and exportable PDF report.
